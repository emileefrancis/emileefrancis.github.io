Project overview

Golden Glow is a beauty, fashion, and self-care website designed for college-aged women, especially users who want inspiration that feels realistic and personal. The site uses a warm color palette, soft language, responsive layout, and simple animation to create a calm browsing experience.

The visual direction is inspired by golden hour, warm yellow energy, earthy colors, and a cool boho style.

Audience persona

Maya is a 20-year-old college student who wants to feel more confident and put together, but she does not want beauty or wellness advice that feels intimidating or unrealistic.

Design goals

Warm

The colors, type, and copy are meant to feel inviting instead of overly polished or cold.

Organized

The navigation separates beauty, fashion, self-care, and guides so users can find content quickly.

Responsive

The layout stacks on mobile and becomes a larger grid on desktop screens.

Code choices

This site uses semantic HTML, one shared CSS file, one shared JavaScript file, a responsive navigation menu, CSS Grid, media queries, hover states, and a small scroll animation.
