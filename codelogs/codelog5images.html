Images

Images are used in websites to add visual information and improve the design of a page. In web development, images can be displayed using HTML or CSS depending on the purpose of the image.

Using the img tag vs background-image

The HTML <img> tag places an image directly into the page as content. This means it behaves like other elements in the layout and can include alt text for accessibility. 

<img src="cake.jpg" alt="Bavarian cream sponge cake">

Using background-image in CSS is different because the image becomes part of the styling of an element instead of being actual page content. 

.box {
  background-image: url("cake.jpg");
}

The main difference is that the img tag is used when the image is meaningful content on the page, while background images are usually decorative. Another difference is layout: an img element affects the page structure, while a background image stays inside the element it is applied to.

Other CSS Properties for Background Images

Besides background-image, other CSS properties control how background images appear.

background-size controls how large the image appears inside the element. 

background-size: cover;

background-repeat controls whether the image repeats across the background. 

background-repeat: no-repeat;

background-position controls where the image is placed inside the element. 

background-position: center;

Dimensions vs Resolution

Dimensions describe the width and height of a bitmap image in pixels, while resolution describes how much visual detail is contained within those dimensions.

Image File Formats

GIF supports animation but has limited colors, so it works best for simple graphics or short animations.

PNG keeps high image quality and supports transparency, which makes it good for logos and graphics.

JPG/JPEG is commonly used for photographs because it compresses images well and keeps file sizes smaller.

WebP is a newer format that can keep good image quality while making file sizes smaller, which is useful for websites.

SVG is different because it is a vector format, meaning it can scale to any size without losing quality. It is often used for icons and logos.

These formats are similar because they all store image data, but they differ in compression, transparency support, and whether they are bitmap or vector images. Because of this, some formats are better for certain uses. For example, JPG works well for photos, PNG works well for graphics with transparency, GIF is used for simple animations, WebP is efficient for web performance, and SVG is ideal for scalable graphics.

Examples

Example of an HTML image:

<img src="image.jpg" alt="example image">

Example of a CSS background image:

.box {
  background-image: url("image.jpg");
  background-size: cover;
}
