CSS Color

CSS uses color to change how things look on a webpage.

The property color changes the text color.

background-color changes the background behind something.

border-color changes the color of the border.

CSS can use different types of color values:

  • Named color: red
  • Hex color: #ffffcc
  • RGB color: rgb(0,0,255)
  • HSL color: hsl(200,50%,50%)
